# Fucker

A [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck) interpreter written in Swift.

With the source code committed in Brainfuck.

```bash
swift run Fucker file://./Example/hello_world.bf

swift run Fucker --program "+[----->+++<]>+.++++++++++++..----.+++.+[-->+<]>.-----------..[--->+<]>++.++.+++++++++++.------------.[--->+<]>---.+[->+++<]>.[-->+<]>---.--[--->+<]>-.++++++++++++.--.+++[->+++++<]>-.+[----->+<]>---.------------.+++..+++++.----.[--->+<]>----.+++[->+++<]>+.+++++.---[->+++<]>.[----->++<]>.[++>-------<]>.+[->+++<]>+.++++++++.------.+++++++++++++."
```

### Compiling

`$ ./bin/build`
